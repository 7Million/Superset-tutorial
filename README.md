## Possible Errors Faced:

`No module named MySQLdb`

## Solution:

1: `sudo apt-get install python3-dev libmysqlclient-dev`

2: `pip install mysqlclient`

3: `pip freeze` just to crosscheck if mysqlclient module now exists
